# Astronauts
- A simple full stack application
    - Angular 4, TypeScript, Webpack, Bootstrap 3, Ruby on Rails 5, REST API
- Shows all astronauts
- Enables to add a new astronaut
- Allows to delete an astronaut

# Get started
Run client (Angular 4) 
- http://localhost:4200/
```bash
npm install
ng serve
```
Run server (Ruby on Rails 5)
- http://localhost:3000/
```bash
bundle install
rake db:create db:migrate db:seed
rails server
```
